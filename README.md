# MedicalBilling Public

CodaHX Medical Billing Application repository for the JHU Singhal Entrepeneurship Award.

## Installation

Install front-end with npm

```bash
  cd front-end
  npm install
```

Install back-end with pip

```bash
  cd back-end
  pip install requirements.txt
```
